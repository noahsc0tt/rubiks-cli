# Rubiks CLI

A command-line Rubik's Cube timer.

## Installation

```bash
gem install rubiks-cli
```

## Usage

```
rubiks [options]
    -c, --clear                      Clear the screen before starting
    -i, --inspection                 Include inspection time
    -l, --loop                       Loop forever
    -s, --scramble                   Generate scramble
    -t, --time                       Start timer

Defaults: --scramble, --time
```
